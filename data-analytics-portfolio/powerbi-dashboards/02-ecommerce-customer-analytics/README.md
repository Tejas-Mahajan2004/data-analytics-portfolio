# E-commerce Customer Analytics Dashboard

## Overview
A Power BI dashboard focused on customer behavior, engagement, and churn
analysis to understand customer value and retention patterns.

## Key Features
- Total customers, login frequency, and lifetime value KPIs
- Customer segmentation by age group
- Churned vs active customer analysis
- Engagement vs lifetime value comparison
- Membership group analysis
- Country-level lifetime value insights

## Business Questions Answered
- Which customer segments are most valuable?
- How does engagement impact customer lifetime value?
- What is the churn distribution across membership groups?

## Tools Used
- Power BI
- DAX
- CSV dataset
