# Mobile Sales Dashboard

## Overview
A Power BI dashboard that analyzes mobile sales performance across brands,
models, payment methods, and time periods.

## Key Features
- Total sales, quantity, and transaction KPIs
- Monthly sales and quantity trends
- Sales distribution by day name
- Payment method analysis
- Brand and mobile model performance
- Interactive filters for deep analysis

## Business Questions Answered
- Which mobile brands and models generate the highest sales?
- How do sales vary by month and day?
- Which payment methods are most preferred by customers?

## Tools Used
- Power BI
- DAX
- Excel dataset
